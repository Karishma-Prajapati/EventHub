
# 🎉 EventHub - Event Management System

A full-stack Event Management System built using PHP, MySQL, JavaScript, HTML, and CSS. The platform allows administrators to create and manage events while users can browse events, register online, receive email confirmations, and download QR-based tickets.

## 🚀 Features

### 👨‍💼 Admin Panel

* Secure Admin Login
* Create, Edit, and Delete Events
* Upload Event Images
* Manage Event Capacity
* View Registered Users
* Monitor Event Registrations

### 👤 User Panel

* User Registration & Login
* Forgot Password with OTP Verification
* Browse Available Events
* Search Events by Name or Location
* Event Registration
* View Registered Events
* Cancel Registrations

### 🎟️ Ticket Management

* Automatic Ticket Generation
* Unique Ticket Code for Every Registration
* QR Code Ticket Support
* Download Ticket Feature

### 📧 Email Notifications

* Registration Confirmation Emails
* OTP Verification Emails
* Password Reset Emails
* PHPMailer Integration

### 🔒 Security Features

* Session-Based Authentication
* Password Encryption
* Prepared Statements
* Input Validation
